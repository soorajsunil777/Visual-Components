# Transfer System (TS1 Evo) Component Library
This folder documents the development of the TS1 Evo (Transfer System 1) component library for Bosch Rexroth modular conveyor systems. 

## Component Modelling

The component modeling process involved systematic design and implementation of TS1 Evo transfer system components and variants based on Bosch Rexroth specifications.
